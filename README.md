# 📈 E-Commerce Sales Dashboard

## 🔹 Overview
Developed an interactive Power BI dashboard to analyze e-commerce sales performance, customer trends, and profit metrics.

## 🔹 Tools & Technologies
- Power BI
- Data Modeling
- DAX (Basic)

## 🔹 Key Features
- Dynamic KPIs for sales and profit tracking  
- Interactive filters and slicers  
- Customer and payment trend analysis  

## 🔹 Insights
- Identified top-performing products  
- Analyzed sales trends over time  
- Helped in understanding customer behavior  

## 🔹 Outcome
This project highlights strong data visualization and dashboard-building skills for business intelligence.
